В Main.java:
* @SpringBootApplication - запускает всё приложение

В контроллерах:
* @Controller - Указывает, что класс является контроллером в приложении 
* @RequestMapping - Используется для маршрутизации
— связывает URL с определённым методом контроллера.
* @GetMapping - это аннотация, которая используется для обработки HTTP-запросов типа GET
* @PostMapping - Принимает данные из форм когда пользователь нажимает "Отправить".
* @PathVariable - Берёт число/слово из адреса страницы и передаёт его в метод.
* @RequestParam - Берёт параметры из строки запроса — той части URL, что после ?.
* @ModelAttribute - "Упаковывает" данные из формы в Java-объект.
* @Valid - Проверяет, правильно ли заполнена форма, прежде чем сохранять данные.

В сервисах:
* @Service - делает класс сервисом
* @Autowired - Автоматически подключает нужные классы друг к другу.
* @Transactional - Включает транзакции БД. Гарантирует, что операция 
с базой данных либо выполнится полностью, либо не выполнится вообще.

В репозиториях:
* @Repository - делает интерфейс репозиторием
* @Query - Позволяет писать свои SQL-запросы, когда стандартных методов не хватает.
* @Param - Связывает параметры метода с параметрами в @Query.

В сущностях:
* @Entity - Создает таблицу
* @Table - Задаёт имя таблицы в базе данных для @Entity класса.
* @Id - Помечает поле как "главный ключ" таблицы — уникальный идентификатор записи.
* @GeneratedValue - Автоматически генерирует уникальные номера ID для новых записей.
* @Column - Настраивает, как поле класса будет храниться в колонке таблицы базы данных.

* @ManyToOne/@OneToMany - создаёт связи между таблицами
* @NotBlank/@NotNull/@Size - проверяет данные при вводе
* @Min/@Max - проверяет числа
* @Override - переопределяет toString()
